
Entity Framework ile ORM yapısı

=>1 Ado.Net bir ORM aracıdır
=>2 Model ORM aracıdır
=>3 Code First ORM aracıdır
=> Bazı firmalar kendi ORM framworklerini geliştirir.

1-Model ile db işlemleri=> Hazır Db ye erişip kullanılır
2- Code First ile db işlemleri=> DB manuel kod ile oluşturulur ve bütün işlemler Code first ile devam edilir
